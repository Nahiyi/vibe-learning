# vibe-learning.skill

一个 AI 驱动的编程导师技能，帮助开发者在使用 AI 辅助编程（vibe coding）的过程中建立真正的技术实力，把 AI 当作学习工具而非拐杖。

## 这是什么？

这是一个 Claude Code 技能，它改变了你用 AI 学编程的方式。它不是让 AI 替你写代码你只管复制粘贴，而是让你成为主导者 —— AI 是你的导师，引导你真正理解，而不是仅仅完成任务。

## 核心教学方法

**控制变量法** -- 当你写的代码不工作，或者你看到 AI 生成的"最佳实践"代码时，这个技能会逐个变量拆解差异，讲解底层机制，帮助你真正理解为什么这样改是对的。

**认知负荷管理** -- 不是每个知识点都值得同等深度。这个技能将知识分为两类：
- **原理类** -- 值得深入理解的核心概念（如生命周期、状态隔离、并发安全）
- **记忆类** -- 配置细节和 API 拼写，可以安全地交给 AI

**苏格拉底式引导** -- 用反问句而非直接给答案。不是直接说"你应该用 useState"，而是问"你觉得 UI 没更新，React 是怎么知道数据变化了的？"

**拒绝代写** -- 绝不会一上来就给你完整可运行的代码。一定会先要求你提供伪代码或思路，再在你的基础上引导。

## 适用场景

在以下场景中激活此技能：
- 想深入学习一门新技术或框架
- 调试自己写的代码，理解为什么出错
- 理解 AI 生成的代码，而不是盲目复制
- 在做项目的过程中真正掌握技术

## 安装方式

将仓库目录的 `vibe-learning` 文件夹（含有`SKILL.md`文档的文件夹）放入 Claude Code 的 skills 目录：

```
~/.claude/skills/vibe-learning/
```

然后在对话中使用 `/vibe-learning` 加载。

```
vibe-learning/

```

## 核心原则

1. **授人以渔** -- 教人捕鱼，不只是送鱼
2. **做中学** -- 你写代码，AI 引导你的理解
3. **知道何时该深挖** -- 原理值得深入，配置只需接受
4. **测了才算真正掌握** -- 可选的微型测试验证真实理解

## 适用人群

具备计算机基础、希望通过真正理解而非 AI 辅助复制粘贴来提升自己的开发者。覆盖全栈：JavaScript、Vue3、React、Java、Spring、Go 等。
